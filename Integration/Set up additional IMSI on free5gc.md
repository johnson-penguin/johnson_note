title: Set up additional IMSI on free5gc

---

## Execute Free5GC
- Command:
```bash=
cd ~/free5gc/
./run.sh
```

## Execute Web Console
- Command:

```bash=
cd ~/free5gc/webconsole
go run server.go
```

![image](https://hackmd.io/_uploads/Sy8kbgKU1e.png)




## Web Console Information:
```
URL: http://127.0.0.1:5000 or http://<Free5GC_IP>:5000
Default User: admin
Default Password: free5gc
```
